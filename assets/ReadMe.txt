Moles Attack Game Assets

Moles Attack game assets for 2D whack a mole smash and hit action fun game.<br>
The game concept is hit the moles, avoid obstacles and complete the missions on each levels.<br>
The files are AICS, EPS10 and transparent PNG.<br>
Separated by layers and it’s folders.<br>
<br> 
File includes:<br>  
- Mole, Bunny, and Hammer character template.<br>  
- Spritesheets samples for those characters.<br>  
- Customizable tile set and background.<br>    
- Miscellaneous items heart, coin, sparkles, and smokes.<br>  
- And simple GUI set.<br>
<br>
